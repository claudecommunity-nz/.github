## Kia ora 👋

We're the home for everyone building with **Claude** across New Zealand — from Auckland to Wellington and everywhere between. Developers, designers, founders, students, hobbyists, and the plain curious: whether you ship AI into production every day or you've never written a line of code, there's a place for you here.

We run local events, share what we're learning, and connect Kiwi builders with each other and with the Claude team.

👉 **Everything starts at [claudecommunity.co.nz](https://claudecommunity.co.nz)**

## Who it's for

**Everyone.** You don't need to be a developer — if you're curious about building with AI, you belong here.

## About the Claude Community Ambassador program

The [Claude Community Ambassador program](https://claude.com/community/ambassadors) empowers local leaders to **build and lead the Claude community in their city**. Ambassadors host meetups, workshops, and hackathons, run live demos, and partner with the Claude team to help shape the future of Claude.

The program supports ambassadors with event funding, ready-to-use content, swag, monthly API credits, access to pre-release features, and a private community alongside the Claude team. It's global, open to people from any background, and welcomes multiple ambassadors per city.

[**Learn more about the program →**](https://claude.com/community/ambassadors)
